# Skillbox_Homework_04

В ветке `Matrix-multiplication` реализовано умножение матриц на число, умножение двух прямоугольных матриц. Если количество строк в первой матрице не равно количеству столбцов во второй матрице, то выводится соответствующее сообщение.

В Ветке `Pascal-triangle` реализован вывод треугольника Паскаля.
